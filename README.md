# 🎨 Unlimited Colors Project
A simple JavaScript project that generates random background colors every second.
## 🚀 Live Demo
https://krishnaydv3214-coder.github.io/BASIC_PROJECT_JS_1/


## ✨ Features
- Random HEX color generation
- Start button to begin color changing
- Stop button to freeze current color
- Smooth UI with responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
